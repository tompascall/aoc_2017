# Playing around with Advent of Code (2017)

[Advent of Code](http://adventofcode.com/2017)

## Ho to get the solutions

```sh
node demo/demo.js days/<dayfile> <input>
```

## Development

### Install dependencies

```sh
npm i
```

### Run tests

```sh
npm run test
```

```sh
npm run test:watch
```

### Eslint

```sh
npm run lint:watch
```
